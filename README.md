# checksums-verificator
утилита для проверки контрольных сумм файлов json и xml

общая интсрукция:

ВВЕДЕНИЕ

программа состоит из двух .py файлов, их надо использовать поэтапно. изначально вы можете накидать произвольный текст в n количество текстовиков, которые вы для удобства можете создать непостредственно внутри папки с двумя .py файлами.

ИСПОЛЬЗОВАНИЕ

после того как вы создадите все нужные вам текстовики вам следует использовать powershell или консоль cmd, в которых вы будете использовать как пример следующую команду

python "get_checksums_here.py" "file1.txt" "file2.txt" "file3.txt"

где file1.txt и так далее это примеры названия файлов
после того как вы получите свои итоговые суммы вы должны их записать в json файл, как пример выглядить это должно так

{
  "file1.txt": "00000000",
  "file2.txt": "00000000",
  "file3.txt": "00000000"
}

теперь прописываем команду:

python tinker.py xdxdxd.json

готово! теперь вы сможете понять совпали ваши контрольные суммы или нет
